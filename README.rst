Requirements
-------------
* install python3-tk
* qtchooser -run-tool=designer -qt=5
* pyuic5 -x simulacra_window.ui -o simulacra_ui.py

Installation
-------------
* pip3 install -r requirements.txt
* python3 setup.py install
